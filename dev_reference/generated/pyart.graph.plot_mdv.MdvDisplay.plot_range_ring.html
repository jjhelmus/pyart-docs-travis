pyart.graph.plot_mdv.MdvDisplay.plot_range_ring¶

MdvDisplay.plot_range_ring(range_ring_location_km, npts=100, ax=None)[source]¶

Plot a single range ring.

Parameters:

range_ring_location_km : float

Location of range ring in km.

npts: int :

Number of points in the ring, higher for better resolution.

ax : Axis

Axis to plot on. None will use the current axis.
