pyart.graph.plot_rsl.RslDisplay¶

class pyart.graph.plot_rsl.RslDisplay(rslfile)[source]¶

A display object for creating plots from data in a RslFile object.

Parameters:

rslfile : RslFile

RslFile object to use for creating plots.

time_begin (datetime) Time of first radar scan.
radar_name (str) Radar name.
rslfile (RslFile) RSL radar object use to create plots.
